# Daily Log Template

**Template for daily conversation logs**

---

```markdown
# Daily Log — YYYY-MM-DD

## Date & Time
- Date: YYYY-MM-DD
- Day: [Monday/Tuesday/etc.]

## Agents Active Today
- [agent name] — [role]
- [agent name] — [role]

## Summary
[Brief summary of what happened today. What was discussed, decided, and accomplished.]

## Conversations

### [Agent Name] × Bobby
**Time:** HH:MM
**Topic:** [what was discussed]

**Bobby:** [verbatim quote from Bobby]
**Agent:** [verbatim response or action taken]

---

### [Agent Name] × Bobby
**Time:** HH:MM
**Topic:** [topic]

**Bobby:** [quote]
**Agent:** [response]

---

## Decisions Made Today
- [ ] [Decision 1]
- [ ] [Decision 2]

## Pending Items
- [ ] [ ] [Item] — Owner: [agent]
- [ ] [ ] [Item] — Owner: [agent]

## Notes
[Anything worth remembering for tomorrow or future reference]

---

_Logged by: [Agent Name] at HH:MM_
```

---

## Usage Notes

1. **Every agent writes to daily-logs/** — No exceptions
2. **Use 3rd person** — "Bobby wants" not "you want"
3. **Be verbatim** — Don't summarize Bobby's words, quote them
4. **Timestamps** — Include time for every conversation block
5. **One block per conversation** — Even short exchanges get their own block
6. **End with pending items** — So nothing falls through the cracks

---

_This log is Bobby's memory. Write it like it matters._
