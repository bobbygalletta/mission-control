## How to Use This Wiki

### Finding Info
1. **Search keywords:** Check `search-index/keywords.md`
2. **Browse by topic:** Use links above
3. **Jump to related:** Every file links to 3+ others

### Adding Info
1. Find the right wiki file (or create one)
2. Update with new info
3. Update search-index/keywords.md
4. Log in today's daily-logs

### Proposing Changes
1. Write proposal to `pending/[change-name].md`
2. Include: What, Why, Impact, Status
3. Wait for Bobby's approval before implementing
