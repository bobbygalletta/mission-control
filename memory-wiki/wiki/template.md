---

## Formatting Rules

1. **Headers:** Use ## for main sections, ### for subsections
2. **Bullet points:** Use - for lists, not numbers (unless sequential)
3. **Links:** Use [[filename]] for internal wiki links
4. **Bold:** Use **text** for emphasis on key terms
5. **Keep it scannable:** Short lines, lots of whitespace

## Cross-Linking Rule
Every wiki file MUST link to at least 3 other wiki files at the bottom.

Minimum required links at bottom of every file:
```
**See also:** [[file-1]], [[file-2]], [[file-3]]
```

## Last Updated
Always include:
- **Last updated:** YYYY-MM-DD
- **by:** [Agent name who made the update]
